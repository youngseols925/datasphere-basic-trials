# 20. SAP Analytics Cloud Reporting 소개 (Introduction)

**소요 시간:** 약 5분

## 학습 목표

SAP Analytics Cloud(SAC)의 기본 개념과 SAP Datasphere와의 연계 방식을 이해합니다.

## 주요 내용

**SAP Analytics Cloud**는 기업의 미션 크리티컬 애플리케이션과 데이터 소스를 최대한 활용할 수 있도록 설계된 통합 분석·계획 솔루션입니다. SAP BTP 내의 분석/계획 솔루션으로서 전사적인 인사이트 확보와 통합 계획 수립을 지원합니다.

### 이 단원에서 학습할 내용

1. **[Optional]** SAP Datasphere 로그온
2. **[Optional]** Analytic Model 임포트
3. SAP Analytics Cloud 로그온
4. Story 생성
5. 차트(Chart)로 데이터 시각화
6. 지도(Geo Map)로 데이터 시각화
7. 테이블(Table)로 데이터 시각화
8. Linked Analysis 및 Input Controls 활용

### 시나리오 개요

이 단원에서는 SAP Analytics Cloud 개요를 학습한 후, SAP Datasphere에 대한 기존 **Live Data Connection**을 활용하여 Analytic Model에 접근합니다.

- **Live Connection**을 통해 SAP Datasphere에서 모델링된 데이터를 직접 대상으로 Story 생성
- 연도별·판매 조직별 매출 총이익(Gross Sales) 분석
- 비즈니스 파트너별 평균 지출 비교

> 💡 Analytic Model을 아직 생성하지 않은 경우, [Optional] 단원을 먼저 진행하세요.
